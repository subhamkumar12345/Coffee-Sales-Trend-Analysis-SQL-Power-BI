# ☕ Coffee Sales Trend Analysis | SQL & Power BI

## 📌 Overview

This project analyzes coffee shop sales data using SQL and Power BI to uncover sales trends, customer purchasing behavior, product performance, and store-wise revenue insights. The interactive dashboard enables data-driven decision-making for inventory management, pricing strategies, and business growth.

---

## 🎯 Objective

The main objectives of this project are:

- Analyze sales performance across products and stores
- Identify top-selling products and categories
- Track monthly and daily sales trends
- Understand customer purchasing patterns
- Build an interactive business intelligence dashboard
